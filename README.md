# ATS_ScreenShots
A little screenshot automation with AutoHotKey to taking screenshot in ATS with Toast

## Hotkeys

|  hotkey        |        description        |   Variable  |
|  ------------  |  -----------------------  |  ---------  |
|  **=**         |  start taking screenshots |  runscript  |
|  **-**         |  stop taking screenshots  |  stopscript |
|  **?**         |  show an helper box       |  help       |
|  **Esc**       |  close helper box         |  closehelp  |
|  **Ctrl+Del**  |  quit the script          |  can't change |

Can change the hotkeys in ***Variables***

All hotkeys: https://www.autohotkey.com/docs/v2/KeyList.htm
